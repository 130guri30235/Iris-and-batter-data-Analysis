# Iris-and-batter-data-Analysis
Exploratory Data Analysis (EDA) and data visualization on the Iris and Batter datasets using Python, Pandas, NumPy, and Matplotlib.

🏏 Iris and Cricket Data Analysis

📖 Project Overview:

This project focuses on Exploratory Data Analysis (EDA) and Data Visualization using the Iris Dataset and a Cricket Batter Dataset. The analysis was performed using Python and popular data science libraries to discover patterns, trends, and insights from the data.

🎯 Objectives:

- Explore and understand dataset structures.
- Perform Exploratory Data Analysis (EDA).
- Generate meaningful visualizations.
- Identify trends and patterns in data.
- Strengthen practical data analysis skills.

🛠️ Technologies Used:

- Python
- Google Colab
- NumPy
- Pandas
- Matplotlib

📊 Analysis Performed:

 Iris Dataset

- Data exploration and inspection
- Statistical summary generation
- Species comparison analysis
- Visualization of petal and sepal measurements

🏏 Cricket Batter Dataset

- Performance data exploration
- Analysis of runs and batting statistics
- Trend identification through visualizations
- Statistical insights generation

🔍 Key Findings:

- Different Iris species exhibit distinct feature patterns.
- Visualizations help reveal relationships between dataset variables.
- Cricket batting data provides insights into player performance trends.
- EDA helps uncover useful information before applying Machine Learning models.
